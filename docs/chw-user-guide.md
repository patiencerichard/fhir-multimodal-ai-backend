# CHW User Guide / Mwongozo wa Mtumishi wa Afya

## Recording Voice Symptoms / Kurekodi Dalili kwa Sauti

1. Tap the microphone icon / Bonyeza ikoni ya kipaza sauti
2. Speak clearly about the patient's symptoms / Eleza dalili za mgonjwa kwa uwazi
3. You can speak in any supported language / Unaweza kuzungumza lugha yoyote inayotumika
4. Maximum 3 minutes / Upeo wa dakika 3
5. Tap stop when finished / Bonyeza simama ukimaliza

## Camera Vitals / Vipimo vya Kamera

1. Ask patient to sit still / Mwambie mgonjwa akae kimya
2. Ensure good lighting on face / Hakikisha mwanga mzuri usoni
3. Hold phone 30cm from face / Shika simu sm 30 kutoka usoni
4. Record for 15 seconds / Rekodi sekunde 15
5. **These are ESTIMATES only** / **Hizi ni MAKADIRIO tu** — confirm with pulse oximeter

## Cough Samples / Sampuli za Kikohozi

1. Record 5 separate coughs / Rekodi kikohozi 5 tofauti
2. Each cough should be clear / Kila kikohozi kiwe wazi
3. Avoid background noise / Epuka kelele za mazingira
4. Wait for green checkmark / Subiri alama ya kijani

## When Offline / Ukiwa Nje ya Mtandao

- The app still works! / Programu bado inafanya kazi!
- You will see "OFFLINE MODE" banner / Utaona bango la "HALI YA NJE YA MTANDAO"
- Triage uses basic rules (not AI) / Uchunguzi unatumia sheria za msingi
- Data syncs automatically when internet returns / Data itasawazishwa mtandao ukirudi
- **If EMERGENCY: call 112 immediately** / **Dharura: piga 112 mara moja**

## Understanding Results / Kuelewa Matokeo

| Category | Meaning | Action |
|----------|---------|--------|
| 🚨 EMERGENCY | Life-threatening / Hatari ya maisha | Call 112 NOW / Piga 112 SASA |
| 🏥 URGENT | Needs doctor today / Anahitaji daktari leo | Go to facility / Nenda hospitali |
| ⚠️ ROUTINE | See doctor soon / Muone daktari hivi karibuni | Schedule visit / Panga ziara |
| ✅ SELF-CARE | Home care OK / Huduma ya nyumbani | Follow advice / Fuata ushauri |

## Escalation / Rufaa

Always refer to facility if:
- EMERGENCY result
- SpO2 below 94%
- TB risk score above 60
- Pregnant woman with any danger sign
- Child under 2 with fever + breathing difficulty
